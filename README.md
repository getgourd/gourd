# gourd
错误汇报、分析、跟踪工作台


## 兼容性要求
- ie系列8及以上
- Chrome
- 火狐
- 其他主流浏览器
- 小程序

## 技术
- 尽量不是用es5/es6的api，语法可以用
- 使用回调方式，不使用promise，保证兼容性
